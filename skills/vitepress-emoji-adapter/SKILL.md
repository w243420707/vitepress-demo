# VitePress Emoji 智能适配 Skill

## 功能说明

此 Skill 用于自动为 VitePress 导航栏标题添加合适的 emoji 图标，基于内容语义智能识别匹配。

## 自动识别规则

AI 将根据以下维度自动选择 emoji：

1. **关键词匹配**：分析标题中的关键词
2. **语义理解**：理解标题的含义和上下文
3. **类别判断**：判断内容所属类别
4. **一致性原则**：保持同类内容使用相似的 emoji

## 自动适配工作流程

当用户要求添加导航项时，AI 应：

### 1. 分析标题内容

- 提取关键词
- 理解语义含义
- 判断内容类别

### 2. 匹配 Emoji

- 查找映射表中的关键词
- 选择最匹配的 emoji
- 确保语义一致性

### 3. 应用配置

- 自动在标题前添加 emoji
- 保持格式统一：`emoji + 空格 + 标题`

### 4. 验证结果

- 检查 emoji 是否与内容相关
- 确保同类内容使用相似 emoji
- 保持整体风格一致

## 智能识别示例

```
用户：添加"Python 入门教程"
AI 自动识别：
- 关键词：教程、入门
- 类别：教程类
- 匹配 Emoji：📚
- 结果：📚 Python 入门教程
```

### 示例 2：添加新资源

```
用户：添加"开源项目推荐"
AI 自动识别：
- 关键词：开源、项目
- 类别：资源类
```

```
- 匹配 Emoji：🌐

```

## 特殊情况处理

### 1. 多关键词匹配

- 优先匹配更具体的关键词
- 例如："网络配置教程" → 🌐（网络优先）

### 2. 无明确关键词

- 根据上下文和语义判断
- 默认使用通用 emoji：📄（文档）

### 3. 自定义 Emoji

- 用户明确指定时，使用用户指定的 emoji
- 优先级：用户指定 > AI 自动识别

## 一致性检查

AI 在添加导航项后应检查：

- ✅ 同一分组下的项目使用相似的 emoji 风格
- ✅ 避免同一页面出现重复 emoji
- ✅ 确保 emoji 与内容语义相关
- ✅ 保持整体视觉和谐

## 使用方法

当需要为导航栏添加 emoji 时，AI 会自动：

1. 分析标题内容和关键词
2. 查询此映射表
3. 选择最合适的 emoji
4. 应用到配置文件中

## 注意事项

- 保持 emoji 的语义一致性
- 避免过度使用 emoji
- 确保可读性和美观性
- 定期更新映射表以适应新内容
