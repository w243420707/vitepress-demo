91GIT - GitHub中国中継 ​

アクセスURL: https://91git.com/

01.1- プロジェクト概要 ​

GitHub中国中継 —— GitHubリポジトリをワンクリックで検索し、ソースコードとリリースアセットを自動パッケージ化して、ダウンロード用の圧縮ファイルを生成します。

国内のネットワーク環境に特化して設計され、開発者がGitHubプロジェクトリソースに迅速にアクセスできるよう支援します。

01.2- 機能特徴 ​

  • 🔍 キーワード検索 - キーワードでGitHubリポジトリを検索し、Star数でソート
  • 📥 ワンクリックパッケージング - ソースコード + リリースアセットを自動ダウンロードし、二重暗号化圧縮ファイルを生成
  • 🔥 GitHub人気 - 今週新規作成された高StarリポジトリTop 10を表示
  • 📊 サイト人気パッケージ - 当日/当月のパッケージング回数が最も多いリポジトリを表示
  • ⏱️ 自動クリーンアップ - パッケージファイルはダウンロードされない場合3分後に自動削除、サーバースペースを節約

01.3- 技術スタック ​

  • フロントエンド: React + TypeScript + Vite + Tailwind CSS
  • アニメーション: Framer Motion
  • バックエンド: Node.js + Express
  • パッケージング: archiver（ZIP）
  • 暗号化: AES-256 + ZIPパスワード

01.4- ユースケース ​

開発者向け ​

  • GitHubプロジェクトのソースコードを迅速に取得
  • 完全なリリースアセットをダウンロード
  • プロジェクトリソースをパッケージ化して共有

研究者向け ​

  • 研究ツールをバッチダウンロード
  • オープンソースプロジェクトをアーカイブ
  • ローカルリソースライブラリを構築

企業ユーザー向け ​

  • 開発ツールを社内で共有
  • チームリソースの同期
  • オフライン環境へのデプロイ

01.5- セキュリティ ​

  • ✅ 二重暗号化保護
  • ✅ 自動ファイルクリーンアップ
  • ✅ ユーザーデータ収集なし
  • ✅ 安全な転送プロトコル
