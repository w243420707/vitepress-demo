
# 91GIT - GitHub 大陆中转站

**访问地址：** [https://91git.com/](https://91git.com/)

## 01.1- 项目简介

GitHub 大陆中转站 —— 一键搜索 GitHub 仓库，自动打包源码与 Release 资产，生成压缩包供下载。

专为国内网络环境设计，帮助开发者快速获取 GitHub 项目资源。

![alt text](Gemini_Generated_Image_aon409aon409aon4.png)

## 01.2- 功能特性

- 🔍 **关键字搜索** - 输入关键字搜索 GitHub 仓库，按 Star 数排序
- 📥 **一键打包** - 自动下载源码 + Release 资产，生成双层加密压缩包
- 🔥 **GitHub 热门** - 展示当周新创建的高 Star 仓库 Top 10
- 📊 **本站热门打包** - 展示当日/当月打包次数最多的仓库
- ⏱️ **自动清理** - 打包文件 3 分钟未下载自动删除，节省服务器空间
- 🎨 **精美 UI** - React + TypeScript 前端，Framer Motion 动画效果

## 01.3- 技术栈

- **前端**：React + TypeScript + Vite + Tailwind CSS  
- **动画**：Framer Motion  
- **后端**：Node.js + Express  
- **打包**：archiver（ZIP）
